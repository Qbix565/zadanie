1. Kiedy powstal JavaScript(podaj date)

JavaScript powstał w 1995 roku i został zaimplementowany w przeglądarce Netscape Navigator w grudniu tego samego roku.

2. Dlaczego jest to język skryptowy

Jest językiem skryptowym, ponieważ jest interpretowany – nie wymaga kompilacji do kodu maszynowego, a jego działanie można zobaczyć od razu w przeglądarce.

3. Gdzie jest wykonywany JS (klient lub serwer)

JavaScript działa głównie po stronie klienta (w przeglądarce), ale dzięki Node.js może być również wykonywany po stronie serwera.

4. Dlaczego JS to bezpieczne narzedzie

JavaScript jest bezpieczny, ponieważ ma ograniczony dostęp do zasobów komputera użytkownika i działa w środowisku przeglądarki.

5. Czy jest to jezyk obiektowy, jesli tak to dlaczego

Tak, JavaScript jest językiem obiektowym, ponieważ korzysta z obiektów, metod i zdarzeń, a jego struktura opiera się na DOM (Document Object Model).

6. wymień zalety JS:

Zaleta 1: Prostota i łatwość nauki – można szybko zacząć programować.

Zaleta 2: Popularność i ogromna liczba przykładów oraz bibliotek dostępnych w Internecie.

Zaleta 3: Uniwersalność – działa w przeglądarkach, odciąża serwery i umożliwia tworzenie dynamicznych aplikacji.

7. Framework to:

Framework to zestaw narzędzi i bibliotek wspomagających tworzenie aplikacji w JavaScript, np. React, Angular, Vue.

8. Do aplikacji mobilnych mozna uzyc framework

Do aplikacji mobilnych można użyć frameworka React Native, który pozwala tworzyć aplikacje na różne systemy operacyjne.

9. Aplikacje responsywne to

Aplikacje responsywne to takie, które automatycznie dopasowują wygląd i układ treści do rozmiaru ekranu urządzenia.

10. Przepisz linie kodu pod nimi wytlumaczenie jak dziala ten fragment kodu

Ten fragment kodu tworzy przycisk o nazwie „Nowa Strona”, a po jego kliknięciu uruchamiana jest funkcja WinOpen(), która ma otworzyć nową stronę.
